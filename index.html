Coding cheat sheet

Navigating in the Command Line

Use pwd to print the current working directory
Use cd folder_name to move to folder_name (use tab to auto-complete directory names)
Use ls to show list the contents of the current working directory
Use cd .. to move up one directory level

Git Commands

git init sets up a git repository in your current folder
git add --all adds all the changes you've made
git commit -m
git push
git pull
